# christianwestbrook.dev

A React application representing my landing page (under construction).

## Pre-requisites

- Git https://git-scm.com/
- Node.js https://nodejs.org/en
- npm https://www.npmjs.com/

## Usage

- Clone the repository with `git clone https://github.com/christian-westbrook/christianwestbrook.dev.git`
- Navigate into the `frontend/` directory of the cloned repo with `cd christianwestbrook.dev/frontend/`
- Install dependencies with `npm install`
- Build the application with `npm run build`
- Navigate into the build directory with `cd dist/`
- Serve the application with `npx serve`
- View the application by navigating to `localhost:3000` in your browser (or whichever port Node tells you)
